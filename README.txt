■ AI動画広告ジェネレーター - ブラウザで開く方法（ポート9000）

【方法1】起動スクリプトをダブルクリック（推奨）
  1. このフォルダ内の「起動サーバー9000.command」をダブルクリック
  2. ターミナルが開き、自動でブラウザが http://localhost:9000/ を開きます
  3. 終了するときは、ターミナルウィンドウで Control+C を押す

【方法2】ターミナルから手動で起動
  1. アプリ「ターミナル」を開く（Spotlightで「ターミナル」と検索）
  2. 次のコマンドを実行：
     cd ~/ai-ad-generator
     python3 -m http.server 9000
  3. ブラウザで http://localhost:9000/ を開く
  4. 終了するときは、ターミナルで Control+C を押す

※ Cursor内のターミナルではポートが使えないため、Macの「ターミナル」アプリで実行してください。
